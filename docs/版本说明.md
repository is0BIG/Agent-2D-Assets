# Agent-2D-Assets 版本说明

## 1.0 基线

`1.0` 保留此前已经完成的三技能工作流：

- `$generate2dsprite`：直接生成 sprite sheet，完成去背、切帧、对齐、GIF 和元数据导出。
- `$generate2dmap`：生成地图、分层场景、prop pack、碰撞、区域和 Godot 场景草案。
- `$generate2daudio`：生成 UI / 战斗 / 法术 WAV 音效，导出分析 JSON 和 `audio-pack.json`。

适合继续使用 1.0 的场景：

- idle、projectile、impact、简单 FX。
- compact prop pack。
- 静态角色、怪物、道具。
- 对动作连续性要求不高的小循环。

## 1.1 新增

`1.1` 新增 video-motion 动作流程，重点解决“不要强迫图像模型直接画完整角色动作 sprite sheet”的问题。

适合使用 1.1 的场景：

- walk / run。
- attack / cast / charge。
- jump / hurt / death / recovery。
- 需要保持角色身份和动作连续性的参考图动画。

1.1 流程：

1. 准备固定镜头动作视频或 PNG 帧序列，建议使用 `#00FF00` 纯绿色背景。
2. 用 `extract_video_frames.py` 抽取完整画布帧。
3. 用 `make_contact_sheet.py` 生成带编号接触表。
4. 将选中的关键帧编号写入 `frame_indices.json`。
5. 用 `video_to_sprite.py` 导出透明帧、`sprite-strip.png`、`sprite-sheet.png`、`animation.gif`、`checker-preview.png`、`sprite-motion-manifest.json` 和 `godot-import.md`。

关键原则：1.1 不逐帧裁角色再居中，而是保留完整源画布。这样走路跨步、挥剑、后坐力和恢复动作才不会被处理器抹平。
