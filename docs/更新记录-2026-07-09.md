## 处理器增强

### Sprite 处理器

文件：

```text
skills/generate2dsprite/scripts/generate2dsprite.py
```

新增能力：

- `--key-color`：支持指定 chroma key 背景色，默认 `#ff00ff`。
- `--tolerance`：更直观的去背容差参数，作为 `--threshold` 的别名。
- `--edge-feather`：对抗锯齿边缘做半透明软化。
- `--despill`：对边缘洋红污染做反混色处理，减少紫边。
- `--apng`：额外导出 APNG 动画预览。
- `--webp`：额外导出无损 animated WebP 预览。
- `pipeline-meta.json` 会记录 `key_color`、`tolerance`、`edge_feather`、`despill`、`apng`、`webp`。

已修复：

- `--prompt-file` 内容会写入 `pipeline-meta.json`。
- `component-mode=largest` 会真正丢弃小噪点，而不是只裁剪 bbox。
- 网格尺寸不能整除图片尺寸时直接报错，避免静默丢失边缘像素。
- 旧 `build-prompt` 不再强制写入 `NOT cute, NOT round`。

### Prop pack 处理器

文件：

```text
skills/generate2dmap/scripts/extract_prop_pack.py
```

新增能力：

- `--key-color`
- `--tolerance`

## image_gen 输出导入

新增文件：

```text
skills/generate2dsprite/scripts/import_generated_image.py
```

用途：

- 从 `$CODEX_HOME/generated_images` 或 `~/.codex/generated_images` 中找到最新 PNG。
- 复制到当前项目输出目录。
- 保留原始生成图不动。
- 写入 `import-generated-image.json` 记录来源。

示例：

```powershell
python .\skills\generate2dsprite\scripts\import_generated_image.py `
  --output .\outputs\my-run\raw-sheet.png
```

## Provider 抽象

新增文件：

```text
skills/generate2dsprite/scripts/image_provider.py
```

当前支持：

- `local_file`
- `codex_generated`

示例：

```powershell
python .\skills\generate2dsprite\scripts\image_provider.py `
  --provider local_file `
  --source .\somewhere\raw.png `
  --output .\outputs\my-run\raw-sheet.png
```

这个接口为后续接 OpenAI API、ComfyUI、Stable Diffusion 或其他图像来源预留了位置。

## 推送说明

默认 HTTPS 推送曾多次失败，表现为：

```text
Failed to connect to github.com port 443
```

诊断结果：

- DNS 能解析 GitHub。
- TCP 443 在提升权限后可连通。
- 默认 Git HTTPS 链路失败。
- SSH over 443 可连通，但本机未配置 GitHub SSH key。

最终成功推送命令：

```powershell
git -c http.version=HTTP/1.1 -c http.sslBackend=schannel push https://github.com/is0BIG/agent-sprite-forge.git HEAD:main
```
