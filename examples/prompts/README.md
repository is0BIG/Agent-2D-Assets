# 示例提示词总览

这里收录每个 skill 的 5 个可复制示例。示例重点不是一次性生成完整游戏，而是给 Codex 一个清晰的资产目标、输出约束和后处理方向。

| Skill | 示例文档 | 内容 |
| --- | --- | --- |
| `$generate2dsprite` | [generate2dsprite.md](./generate2dsprite.md) | 角色 idle、四方向行走、法术包、impact FX、道具包 |
| `$generate2dmap` | [generate2dmap.md](./generate2dmap.md) | Godot RPG 地图、横版关卡、塔防路线、战术格子、Roguelike 房间 |
| `$generate2daudio` | [generate2daudio.md](./generate2daudio.md) | UI、火球、平台动作、科幻战斗、拾取奖励音效 |

相关真实音频素材位于 [examples/audio](../audio)。
